# 第4章 演習問題 - リクエストとレスポンス

## 問題1: Pydanticモデルの基本
商品情報を表すPydanticモデルを作成し、POST /productsエンドポイントを実装してください。

## 問題2: ネストしたPydanticモデル
ユーザーと住所情報を含むネストしたPydanticモデルを作成してください。

## 問題3: レスポンスモデル
リクエストで受け取ったパスワードをレスポンスから除外するようにしてください。

## 問題4: ステータスコードの使い分け
CRUDエンドポイントで適切なステータスコードを返すAPIを実装してください。

## 問題5: カスタムヘッダー
カスタムヘッダーを読み取り、レスポンスに含めるエンドポイントを作成してください。

## 問題6: Cookieの操作
Cookieを設定・読み取りするエンドポイントを作成してください。

## 問題7: リクエストボディの検証
必須フィールドとオプションフィールドを持つモデルを作成してください。

## 問題8: 複数のボディパラメータ
複数のPydanticモデルを受け取るエンドポイントを実装してください。

## 問題9: ファイルアップロード
画像ファイルをアップロードするエンドポイントを作成してください。

## 問題10: レスポンスの例
OpenAPIドキュメントにレスポンス例を追加してください。
