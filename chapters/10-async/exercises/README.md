# 第10章 演習問題 - 非同期処理とバックグラウンドタスク

## 問題1: 非同期エンドポイント
async defを使った非同期エンドポイントを作成してください。

## 問題2: asyncio.sleep
asyncio.sleepを使って遅延を伴う処理を実装してください。

## 問題3: 非同期HTTP リクエスト
httpxライブラリを使って外部APIを非同期で呼び出してください。

## 問題4: バックグラウンドタスクの基本
メール送信をバックグラウンドタスクとして実装してください。

## 問題5: 複数のバックグラウンドタスク
複数のバックグラウンドタスクを同時に実行してください。

## 問題6: 並行処理
asyncio.gatherを使って複数の非同期処理を並行実行してください。

## 問題7: 非同期データベース操作
非同期でデータベースCRUD操作を実装してください。

## 問題8: ストリーミングレスポンス
Server-Sent Events (SSE) を使ったストリーミングエンドポイントを作成してください。

## 問題9: タスクキュー
Celeryやarqを使ったタスクキューの実装を検討してください。

## 問題10: パフォーマンス比較
同期版と非同期版のエンドポイントのパフォーマンスを比較してください。
