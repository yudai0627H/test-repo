# 第5章 演習問題 - Pydanticとバリデーション

## 問題1: 基本的なPydanticモデル
ユーザー登録のためのPydanticモデルを作成してください（name, email, password, age）。

## 問題2: Fieldによる制約
価格が0より大きく100万以下、名前が3〜100文字の商品モデルを作成してください。

## 問題3: カスタムバリデーター
メールアドレスのドメインが特定のリストに含まれているか検証してください。

## 問題4: 複数フィールドのバリデーション
開始日が終了日より前であることを検証するイベントモデルを作成してください。

## 問題5: ネストしたモデル
ユーザーと住所情報を含むネストしたモデルを作成してください。

## 問題6: Config クラス
orm_mode や alias を使ったモデルを作成してください。

## 問題7: Optional と Union
オプショナルフィールドと複数の型を受け入れるフィールドを持つモデルを作成してください。

## 問題8: リストと辞書のバリデーション
タグ（List[str]）とメタデータ（Dict[str, Any]）を持つ記事モデルを作成してください。

## 問題9: Enumを使った選択肢
ユーザーロール（admin, user, guest）をEnumで定義してください。

## 問題10: カスタムデータ型
電話番号や郵便番号などのカスタムデータ型を作成してください。
