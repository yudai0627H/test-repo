# 第6章 演習問題 - データベース連携

## 問題1: データベースセットアップ
SQLiteデータベースとSQLAlchemyエンジンをセットアップしてください。

## 問題2: ユーザーモデルの作成
id, name, email, created_atを持つUserモデルを作成してください。

## 問題3: Create操作
新しいユーザーを作成するエンドポイントを実装してください。

## 問題4: Read操作
全ユーザーを取得するエンドポイントと、IDで特定のユーザーを取得するエンドポイントを実装してください。

## 問題5: Update操作
ユーザー情報を更新するエンドポイントを実装してください。

## 問題6: Delete操作
ユーザーを削除するエンドポイントを実装してください。

## 問題7: リレーションシップ
UserとPostの1対多のリレーションシップを実装してください。

## 問題8: フィルタリングとソート
名前で検索し、作成日でソートするエンドポイントを実装してください。

## 問題9: ページネーション
ページネーション機能を実装してください（skip, limitパラメータ）。

## 問題10: トランザクション
複数のデータベース操作をトランザクションで管理してください。
