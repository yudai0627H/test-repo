# 第7章 演習問題 - 依存性注入

## 問題1: 基本的な依存関数
データベースセッションを返す依存関数を作成してください。

## 問題2: 共通パラメータの依存関数
skip, limit, sort_byパラメータを持つ共通クエリパラメータ依存関数を作成してください。

## 問題3: 依存関係の階層化
get_current_user -> get_current_active_user の階層化された依存関係を実装してください。

## 問題4: クラスベースの依存関係
ページネーション用のクラスベース依存関係を作成してください。

## 問題5: 依存関係のキャッシュ
use_cache=Trueを使って依存関係をキャッシュしてください。

## 問題6: グローバル依存関係
アプリケーション全体に適用されるグローバル依存関係を設定してください。

## 問題7: サブ依存関係
複数の依存関係を組み合わせた複雑な依存関係を作成してください。

## 問題8: バリデーション依存関係
リクエストヘッダーを検証する依存関係を作成してください。

## 問題9: 依存関係でのエラーハンドリング
依存関係内で適切な例外処理を実装してください。

## 問題10: 依存関係のテスト
依存関係をオーバーライドしてテストを実装してください。
